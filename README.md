# Human Following Robot

## Overview
A human-following robot designed using Arduino and ultrasonic sensors. The robot detects and follows a person while maintaining a safe distance.

## Objectives
- Detect a human target
- Follow the target automatically
- Avoid collisions and maintain distance

## Components Used
- Arduino UNO
- Ultrasonic Sensor
- Motor Driver (L298N)
- DC Motors
- Robot Chassis
- Battery Pack

## Features
- Automatic human tracking
- Distance measurement
- Autonomous movement
- Obstacle awareness

## Working Principle
The ultrasonic sensor continuously measures the distance between the robot and the target. Based on the measured distance, the Arduino controls the motors to move forward, stop, or adjust direction.

## Applications
- Service robots
- Smart assistance systems
- Educational robotics projects

## Future Improvements
- Camera-based tracking
- Bluetooth/Wi-Fi control
- AI-based object detection
